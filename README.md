# Self-Evolving Agent

## Study workspace

The course can be used remotely from a phone and synced back to a home coding session. Deployment and agent-fetch instructions are in [Remote Study Workspace](docs/remote-study-workspace.md).

The learning system uses one manifest and milestone-based state across lessons, implementation sessions, evidence, and review. See [Learning System](docs/learning-system.md).

Start a teaching session with:

```sh
./tools/study-context
```

Run a lesson's configured proof with:

```sh
./tools/prove-lesson 0006-agent-loop-primitive
```
